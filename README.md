# vue-todos

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Runs unit tests and puts coverage report in coverage/lcov-report/index.html
```
npx jest --coverage
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
